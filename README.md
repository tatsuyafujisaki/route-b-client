## What is Route B?

"Route B" is a communication channel provided by Japanese smart meters to transmit electricity usage data directly to in-home devices such as HEMS (Home Energy Management System).
For more details, please refer to [TEPCO's official explanation about Smart Meters](https://www.tepco.co.jp/en/pg/development/domestic/smartmeter-e.html#:~:text=route%20B).
